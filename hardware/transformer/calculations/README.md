# Transformer Calculations

This folder contains the design and verification calculations for the custom RM10/I flyback transformer used in the 5 V offline SMPS project.

## Files

| File | Description |
|---|---|
| transformer_design_analysis.pdf | Final transformer characterization and electrical design summary. Click this directly in GitHub to view. |
| bobbin_fit_estimation.pdf | Bobbin winding-fit calculation for split primary, TIW secondary, auxiliary winding, and insulation stackup. Click this directly in GitHub to view. |
| matlab_results.docx | Raw MATLAB calculation output. GitHub may require download for Word files. |
| matlab_script.docx | Raw MATLAB script document. GitHub may require download for Word files. |
| saturation.xlsx | Raw saturation / magnetic operating-point spreadsheet. GitHub may require download for Excel files. |

## Key Results

| Parameter | Value |
|---|---:|
| Output power target | 5 W |
| Switching frequency | about 132 kHz |
| Primary turns | 119 turns |
| Measured primary magnetizing inductance | 2.476 mH |
| Secondary inductance | 26.64 uH |
| Auxiliary inductance | 26.77 uH |
| Conservative leakage inductance | about 53.10 uH |
| Coupling factor | about 0.989 to 0.991 |
| Estimated primary peak current | about 0.206 A |
| Total bobbin radial build | about 2.177 mm |

## Notes

GitHub does not reliably preview DOCX and XLSX files. The PDFs in this folder are viewable directly; the raw Office files are included for traceability and may need to be downloaded.
