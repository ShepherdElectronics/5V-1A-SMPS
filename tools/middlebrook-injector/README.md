# Middlebrook Loop-Injection Box

Reusable Middlebrook loop-injection box used for future SMPS feedback-loop measurement.

| File / Folder | Description |
|---|---|
| [build-guide.md](build-guide.md) | GitHub-viewable extracted build guide. |
| images/ | Injector photos converted to JPG. |
| waveforms/ | Function generator and injector-box waveform captures. |

Safety: this injector is for low-voltage control loops only, not high-voltage drain or DC-bus nodes.
